SEQ_TABLES TUTORIAL¶

What does the class do:¶

Seq tables is to be used for viewing aligned sequences using pandas dataframes/numpy arrays. It will put lists or series of sequences and, optionally, quality scores into dataframes. Each sequence of tables in which each row is a specific sequence and each column is a specific position within the sequence. This allows useful slicing and determination of hamming distance between sequences

SOME WARNINGS BEFORE USING¶

  1. Use python2 - It does not appear to run as expected on python3, plan to fix that bug
  2. Use firefox - Plotly is not currently displaying plots in ipython notebook on Chrome? Suggest using firefox for now
  3. Avoid converting each individual base/residue into a dataframe (i.e. dont do this => pd.DataFrame(sq.seq_table.view_bases())). In pandas, all stings even characters are treated as objects so the memory usage becomes extremely large (He told me that he will also deprecate being able to define cells as single characters. Instead, use pandas categories if desired but I find that categories are still kind-of buggy. For this reason, most of the variables in the class treat each string using their ascii values

There are multiple ways to initialize this class. Essentially it just needs a list of aligned sequences to one another. If you would like to you could also past in a list of quality scores

seqs = ['AAA', 'ACT', 'ACA']
quals = ['ZCA', 'DGJ', 'JJJ']
sq = seq_tables.seqtable(seqs, quals)

This class has three important attributes

1. sq.seq_df => returns a dataframe of the original sequence and/or quality

sq.seq_df
seqs quals
0 AAA ZCA
1 ACT DGJ
2 ACA JJJ

2. sq.seq_table => this represents all letters in the sequences as a table of rows/columns of ascii values

sq.seq_table
1 2 3
0 65 65 65
1 65 67 84
2 65 67 65

If you would like, you could view the letters as actual letters instead of ascii

sq.view_bases()

array([['A', 'A', 'A'],
       ['A', 'C', 'T'],
       ['A', 'C', 'A']],
      dtype='|S1')

# Now we can slice select positions
sq.view_bases()[:, 1:3]

array([['A', 'A'],
       ['C', 'T'],
       ['C', 'A']],
      dtype='|S1')

3. sq.qual_table => this represents the qualitysequence as actual quality bases

sq.qual_table
1 2 3
0 57 34 32
1 35 38 41
2 41 41 41
